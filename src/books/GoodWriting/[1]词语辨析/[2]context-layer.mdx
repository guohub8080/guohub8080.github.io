# 语境色彩：适度与过度

“适度”是范围词，暗示可以调节；“过度”是状态词，强调已经超过界限。写作里把这两个词放在一起，能够制造出“可控 vs 失控”的对比。

## 如何运用

| 场景 | 更贴切的词 |
| --- | --- |
| 政策建议 | 适度，引导读者思考调节空间 |
| 反面案例 | 过度，突出反常情形 |
| 产品文案 | 适度，给人“可自定义”的自由感 |

> “适度惊喜”暗示你仍然掌控节奏；  
> “过度惊喜”则隐含作者认为“有点太多了”。

## 模板句

1. **适度 + 名词**：适度留白、适度提醒、适度扩写。  
2. **防止 + 过度 + 动词**：防止过度加工、防止过度解释。  
3. **适度 A，避免过度 B**：适度铺陈背景，避免过度堆砌术语。

## 快速练手

写一个 80 字以内的段落，用“适度”描述作者的安排，再用“过度”提醒读者潜在风险。完成后删去任意一个词，观察语气是否失衡。
