# Claude Code 快速入门

本文档将指导您快速开始使用 Claude 进行代码开发。

## 基础用法

### 1. 代码生成

描述您需要的功能，Claude 将生成相应的代码。

```python
# 示例：生成一个计算阶乘的函数
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n-1)
```

### 2. 代码解释

提供代码，Claude 可以解释其功能。

### 3. 代码优化

Claude 可以帮助优化现有代码，提高性能和可读性。
