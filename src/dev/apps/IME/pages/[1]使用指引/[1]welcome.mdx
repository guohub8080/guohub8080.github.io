## 系统简介

欢迎使用输入法词库备份系统！本系统旨在帮助您安全地管理和备份输入法词库，确保您的个性化词汇和输入习惯数据不会丢失。

## 技术架构

本系统采用前后端分离的架构设计：

- 前端：基于 React 构建的现代化用户界面
- 后端：基于 Python FastAPI 框架的高性能 RESTful API 服务

## 部署说明

### 后端部署

后端服务需要单独部署。由于后端涉及数据库配置、API 密钥等敏感信息，我们将其托管在独立的 GitHub 私有仓库中。

部署步骤：
1. 从独立仓库克隆后端代码
2. 配置环境变量和数据库连接
3. 安装 Python 依赖
4. 启动 FastAPI 服务

### 前端配置

前端应用需要配置后端 API 地址，确保能够正常与后端服务通信。
